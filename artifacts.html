Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide

EMR-LIS Encounter

Encounter resource associated with the Service Request for a lab order generated by the EMR
EMR-LIS Service Request

A service request created by an EMR representing an order for an LIS system
EMR-LIS Observation

Observation resource created by a LIS that representing a result for an order submitted by an EMR System
EMR-LIS Workflow Patient

A patient used for EMR - LIS lab order and result communication.
EMR-LIS Diagnostic Report

DiagnosticReport resource created by a LIS that representing results of an order submitted from an EMR System
EMR-LIS Workflow Task Resource

A Task resource for tracking order state and synchronizing communication between an EMR and a LIS
