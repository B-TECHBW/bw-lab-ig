## Building Locally

### Install nodejs
https://nodejs.org/en/download/

### Install Sushi
```
npm install -g fsh-sushi
sushi --help
```

### Update IG Publisher
```
./_updatePublisher.sh
```

### Run IG Publisher
```
./_genonce.sh
```
